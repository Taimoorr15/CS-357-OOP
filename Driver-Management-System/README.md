# Driver-Management-System
This is a Driver Management System which keeps track of all the drivers and their authenticity.

Overview

The Driver Management System is a mini Object-Oriented Programming (OOP) project built in Python.
It models real-world entities like Person, Driver, License, and Car using OOP concepts such as inheritance, composition, and association.

This project demonstrates the use of:

Inheritance → Driver inherits from Person.

Composition → Driver has-a License.

Association → Driver drives a Car.
Classes and Relationships

Person

Base class with attributes name, age, and CNIC.

Driver

Inherits from Person.

Has a License object (composition).

Associated with Car.

License

Created for a Driver, contains license details.

Car

A vehicle that can be associated with a Driver.
Concepts Used

OOP Principles: Inheritance, Composition, Association
